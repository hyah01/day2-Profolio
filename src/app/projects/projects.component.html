Human VS Goblin

This project is a GUI-based RPG titled "Humans vs Goblins" created using Java Swing. It features a grid-based game world where players control human characters to move, interact, and engage in turn-based combat with goblins. The game includes a detailed inventory system, equipment that modifies player stats, and random treasure chests with collectible items. The combat system is turn-based and provides visual feedback, with loot drops and rewards enhancing gameplay.
